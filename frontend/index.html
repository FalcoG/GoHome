Instructions

To connect with Home Assistant you must provide a long-lived access token.

In order to create such an access token, follow these instructions:

  1. Navigate to Home Assistant web interface
  2. Click on your user profile
  3. Go to "Security"
  4. Navigate to "Long-lived access token"
  5. Create a token with a recognizable name
  6. Copy the token and paste it into GoHome
